Esse trabalho foi desenvolvido durante as disciplinas de Aprendizado de Máquina Supervisionado e Mineração de Dados. 

Foi utilizado uma base de dados real extraído de uma empresa atuante na área da Saúde, fato que traz uma credibilidade maior ao trabalho desenvolvido.

As avaliações foram feitas baseadas no score da competição Kaggle: https://www.kaggle.com/competitions/auditorias-em-planos-de-sade-ufg-fase-2/overview

Exercitei os conhecimentos aprendidos desde o entendimento dos dados até sua modelagem, com todos os processos comentados e devidamente documentados para fácil abstração.
